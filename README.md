## 📚 Sobre

Estudo prático dos principais recursos do ECMAScript antes de começar com um framework.

Cada tópico possui uma pasta com:

## 🚀 Como usar

Para navegar entre as pastas no terminal:

```bash
cd JavaScript-Moderno # ou cd Metodos-Array
cd nome-da-pasta
node index.js
```
